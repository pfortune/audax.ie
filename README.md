# audax.ie

Public issue tracker for [audax.ie](https://audax.ie), a ride tracking and award progress tool for the Irish Audax cycling community.

## Reporting a Bug

Found something that isn't working right? [Open an issue](https://github.com/pfortune/audax.ie/issues/new) and include:

- What you were doing when it happened
- What you expected to happen
- What actually happened
- Your browser and device (desktop/mobile)
- A screenshot if it helps

## Feature Requests

Have an idea for something that would make the tracker better? Feel free to [open an issue](https://github.com/pfortune/audax.ie/issues/new) with a description of what you'd like to see.

## Note

This repository is for issue tracking only. The application source code is not hosted here.

## Contact

For general questions or support, visit the [Support page](https://audax.ie/support) on the app.
